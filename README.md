# Hate_Music_Bot
## 簡單容易使用的DC音樂bot
透過wavelink串流音樂，並使用discord.py串接Discord:
- 單一斜槓指令
- UI控制面板
- 無前綴指令
- 播放歌曲不會造成洗版
- 播完歌曲自動離開並刪除控制面板
- 可以訂閱頻道來獲取直播/新影片通知
